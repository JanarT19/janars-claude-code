# Quick Start: Publish Your Plugin in 5 Minutes

## TL;DR

```bash
# 1. Create GitHub repo at: https://github.com/new
# Name: janars-claude-code
# Visibility: Public

# 2. Push your code
cd ~/Documents/GitHub/janars-claude-code
git remote add origin https://github.com/JanarT19/janars-claude-code.git
git push -u origin main

# 3. Share with others
# Installation commands:
# /plugin marketplace add JanarT19/janars-claude-code
# /plugin install janars-setup@janars-claude-code
```
